    <h3>A cli to watch anime with Turkish subtitle.</h3>
</div>

## Building

1. Clone this repository
   - `git clone https://github.com/0x15d3/turk-ani-cli.git`
2. Run:
   - `bun run src/index.ts --anime "One Piece"`
